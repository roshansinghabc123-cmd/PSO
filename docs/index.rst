π-Stack Optimizer Documentation
===============================

The **π-Stack Optimizer** is a high-performance computational framework for identifying energetically favorable stacking configurations in molecular systems (e.g., dimers, trimers). It integrates semi-empirical quantum chemistry methods (xTB) with robust global optimization algorithms like Particle Swarm Optimization (PSO), Genetic Algorithms (GA), and Grey Wolf Optimizer (GWO).
