Usage and CLI Reference

The π-Stack Optimizer provides a robust command-line interface.

Main Command: pi-stack-generator

Run the optimization using the wrapper script or python module.

pi-stack-generator [INPUT_FILE] [OPTIONS]

General Options

input_file

(Required) Path to the .xyz or .json file describing the system.

--optimizer, -opt

Optimization algorithm to use. * Default: pso * Choices: pso, ga, gwo, pso_nm

--workers, -w

Number of parallel xTB workers. * Default: 4

--max-iters, -n

Maximum number of optimization iterations. * Default: 300

--seed

Random seed for reproducibility. * Default: 42

Algorithm Specific Parameters

PSO Options:

--swarm-size

Number of particles in the swarm. * Default: 60

--inertia

Inertia weight ( \(\omega\) ). * Default: 0.73

--cognitive

Cognitive coefficient ( \(c_1\) ). * Default: 1.50

--social

Social coefficient ( \(c_2\) ). * Default: 1.50

Genetic Algorithm Options:

--ga-population

Population size. * Default: 80

--ga-mutation-rate

Probability of mutation. * Default: 0.10

--ga-crossover-rate

Probability of crossover. * Default: 0.90

Grey Wolf Optimizer Options:

--gwo-pack-size

Number of wolves (search agents). * Default: 50

--gwo-a-start

Initial value of the convergence parameter \(a\). * Default: 2.0

--gwo-a-end

Final value of the convergence parameter \(a\). * Default: 0.0

Physical & backend Parameters

--r-min

Minimum allowed intermolecular distance (Penalty threshold). * Default: 3.0 Å

--clash-cutoff

Hard clash cutoff distance. * Default: 1.6 Å

--gfn

xTB GFN parameterization. * Default: 2

Hyperparameter Tuning (pi-hyperopt)

The pi-hyperopt command is used for automated parameter tuning using Optuna.

pi-hyperopt --molecules-root ./molecules --trials-per-molecule 50 --joint

Arguments:

--molecules-root

Directory containing subfolders of molecules to train on.

--trials-per-molecule

Number of Optuna trials to run per molecule. * Default: 50

--joint

If set, runs Joint Optimization mode to find a single best parameter set for the dataset.
